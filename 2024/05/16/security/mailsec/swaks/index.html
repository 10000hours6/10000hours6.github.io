swaks(SWiss Army Knife Smtp)

获取地址

1、kali自带

2、官网

http://www.jetmore.org/john/code/swaks/

3、yum install swaks

4、apt install swaks

基本用法

swaks -to 被测试的邮箱

参数

–header 邮件头信息，key:value (subject为标题)

–header-X-Mailer “value” 设置Mailer特征，不设置默认是带有swaks标记的（X-Mailer: swaks v20190914.0 jetmore.org/john/code/swaks/）

–header-Message-Id “value” 设置邮件Message-ID

–ehlo value 伪造ehlo头，要伪造发件人的域名，提供身份认证

–body value 邮件正文

–to 邮件接收者，多个接收者使用“,”进行分隔

–from 伪造的发件人

发送模板邮件

原始邮件转存为eml文件，swaks参数–data设置为保存的eml文件。

伪造的邮件和原始邮件不同点：

1、查看邮件原文：Sender不同

​ Sender: hawk0@vict.site

​ Sender: 54b7fc66-3b03-11ec-9a78-5254001bd3a2@postman.huoxian.cn

2、标题可设置不同

其他（包含时间）都是相同的

eml文件：【安全共识】项目进展通知_.eml

示例

swaks –header “Subject: vapd10271613” –header-X-Mailer “staff.domain” –header-Message-Id “staff.domain” –to mail1@staff.domain,mail2@staff.domain –from vap@staff.domain –ehlo staff.domain –body “2021.1028.1421helloword”

swaks –header “Subject:test1” –to nc0isini@foxmail.com –from hawk0@vict.site –header-X-Mailer “vict.site”

swaks –header “Subject:test1” –to nc0isini@foxmail.com –from hawk0@vict.site
