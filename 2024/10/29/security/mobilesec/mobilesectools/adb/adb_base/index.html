安装

下载

工具包下载地址

解压到指定目录

配置

添加环境变量

Windows

Windows path 中添加路径

D:\tools\security\adb\platform-tools

cmd 中执行 adb 命令即可

Mac

命令

获取本地连接设备列表

adb devices

unauthorized 表示设备未授权

device 表示连接成功

连接设备

adb connect ip:5555

进入shell

adb shell

待确认问题，多个设备，默认会连接哪个？

查看应用列表

pm list packages

adb shell pm list packages

语法

pm list packages

参数

-f 显示应用关联的apk文件
-d 只显示disabled的应用
-e 只显示enabled的应用
-s 只显示系统应用
-3 只显示第三方应用
-i 显示应用的 installer
-u 包含已卸载的应用

过滤

参数后直接添加需要过滤的字符

pm list packages -i -u box 包名中包含 box

查看设备信息

查看芯片类型

getprop ro.product.cpu.abi

arm64-v8a === 64位

armeabi-v7a === 32位

查看系统版本

getprop ro.build.version.release

查看系统信息

需要root权限，也就是环境为 #

cat /system/build.prop | grep product

查看系统api版本

查看设备型号

getprop ro.product.mode

获取厂商名称

getprop ro.product.brand

查看设备序列号

getprop ro.serialno

查看mac地址

cat /sys/class/net/wlan0/address

需要root权限，也就是环境为 #

查看内存信息

cat /proc/meminfo

查看屏幕分辨率

adb -s 10.235.19.57:5555 shell “dumpsys window | grep mUnrestrictedScreen”

dumpsys window | grep mUnrestrictedScreen

wm density
