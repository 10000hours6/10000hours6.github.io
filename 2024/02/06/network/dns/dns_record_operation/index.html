dns_record_operation

本文最后更新于 2024-02-07T09:52:22+08:00

背景

对常见的DNS记录及操作学习总结

A记录

描述

主机名称，域名对应的IPv4地址

操作

查询

nslookup domain

nslookup 不指定参数，默认查询A记录

host -t A domain

AAAA记录

IPv6版本的A记录，域名对应的IPv6地址

操作

查询

nslookup -type=AAAA domain

host -t AAAA domain

NS记录

描述

域名服务器记录，也就是该记录指定由那台域名服务器解析域名

操作

查询

nslookup -type=NS domain

#### host -t NS domain

network > dns
