ftp_install

本文最后更新于 2024-04-03T19:59:36+08:00

简介

FTP（File Transfer Protocol）文件传输协议，用于在网络上进行文件传输的协议

安装

Linux

Linux下常用的ftp工具为vsftp

检查是否安装

rpm -qa | grep vsftpd

yum安装

yum install vsftpd -y

操作

启动

service vsftpd start

systemctl start vsftpd

停止

service vsftpd stop

systemctl stop vsftpd

状态

service vsftpd status

systemctl status vsftpd

默认信息

默认可执行文件

/usr/sbin/vsftpd

默认安装配置目录

/etc/vsftpd/

默认核心配置文件

/etc/vsftpd/vsftpd.conf

配置说明

anonymous_enable=YES # 允许匿名登录

local_enable=YES # 允许本地用户登录

write_enable=YES # 允许写操作

local_umask=022 # 新增文件权限，默认022
